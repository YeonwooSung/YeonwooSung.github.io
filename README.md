# YeonwooSung.github.io

* Theme: [Chirpy Theme](https://github.com/cotes2020/jekyll-theme-chirpy/)

## Build and Deploy

```bash
$ bundle exec jekyll build
# or
$ bundle exec jekyll serve

# deploy (for GitHub Actions)
$ bundle lock --add-platform x86_64-linux
```
